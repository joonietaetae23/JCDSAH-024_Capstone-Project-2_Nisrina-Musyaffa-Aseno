# JCDSAH-024_Capstone-Project-2_Nisrina-Musyaffa-Aseno

New York City Taxi and Limousine Commission (TLC) merupakan lembaga yang berperan sebagai regulator untuk mengatur sistem transportasi taksi Medallion, kendaraan sewa, van komuter, serta kendaraan paratransit di Kota New York. Lembaga ini telah didirikan sejak 2 Maret 1971.

Dataset NYC TLC Taxi Trip memiliki total 20 kolom dengan kategori data sebagai berikut:
1. Cost & Payment Information
2. Vendor & Operational Info
3. Trip Identifier
4. Temporal Information
5. Spatial Information
6. Trip Metrics

Karena data memiliki sejumlah kolom null, maka dilakukan data *handling* dengan mengisi data berdasarkan data pendukung lain, baik dari kolom lain maupun berdasarkan median/mean data.

Berdasarkan data yang dimiliki, dilakukan sejumlah analisis sebagai berikut:
- Temporal Demand
- Spatial Hotspots
- Trip Behaviour
